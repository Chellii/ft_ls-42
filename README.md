# ft_ls-42

## Overview
FT_LS is a 42 Project that aims to mimic libc ls function

- **Basic functions**:
  -l, -R, -a, -r, -t.

## Installation
```
$> make
$> ./ft_ls [-Ralrt] [file ...]
```
